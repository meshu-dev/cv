A digital version of my CV.

## Setup

This is a frontend app that requires use of the [Portfolio](https://github.com/meshu-dev/portfolio) API app.

Git clone the Porfolio app and then follow the setup guide to get it up and running.

Create the .env file by copying the .env.example file.

```bash
cp .env.example .env
```

Fill in the environment variable values.

Run the CV app.

```bash
npm run dev
```

It should be available at [http://localhost:3000](http://localhost:3000).
